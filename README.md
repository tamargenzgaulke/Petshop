# Petshop
Site de Petshop para Disciplina de Fundamentos da Web

Petiscão – Sistema Web para Petshop

Descrição do Projeto
O Petiscão é um projeto desenvolvido para a disciplina de Desenvolvimento Web, com o objetivo de criar uma plataforma online simples para um petshop fictício.  
Construir o front-end em HTML, CSS, JavaScript e frameworks, incluindo o formulário de cadastro do cliente e do pet, assim como a forma de agendamento para tele-busca ou entrega do pet no local.
---

Objetivo Geral
Desenvolver uma plataforma web estática para o Petiscão, permitindo a visualização de produtos e serviços do petshop de forma organizada, acessível e intuitiva.

---

Objetivos Específicos
1. Estruturar o site utilizando corretamente as seções de cabeçalho, corpo e rodapé.  
2. Criar três páginas principais: **Início**, **Produtos**, **Rações**, **Acessórios** e **Serviços**.  
3. Exibir ao menos dois produtos de cada categoria obrigatória (Acessórios, Rações e Produtos de Higiene).  
4. Descrever os serviços oferecidos (Banho e Tosa, com e sem tele-busca).
5. Disponibilizar formulário de cadastro do cliente e do pet.
6. Disponibilizar formulário de solicitação de serviços incluindo formas de pagamento.  
7. Publicar o site no **GitHub Pages** com links funcionais.
